2. Información general sobre el tratamiento de datos

Este sitio web prescinde deliberadamente de cualquier forma de seguimiento, análisis, cookies publicitarias o integración de servicios externos. Al acceder al sitio no se transmiten datos personales a terceros.

3. Archivos de registro del servidor

Al acceder a este sitio web, el proveedor de hosting recopila datos técnicamente necesarios en archivos de registro del servidor que su navegador transmite automáticamente: dirección IP, fecha y hora de la solicitud, página consultada, URL de referencia, navegador utilizado y sistema operativo.

La base jurídica es el art. 6 apdo. 1 lit. f RGPD (interés legítimo en el funcionamiento técnicamente correcto del sitio web). Estos datos no se combinan con otras fuentes de datos y se eliminan tras un breve período de tiempo.

4. Alojamiento

Este sitio web está alojado en los servidores de GitHub Pages (GitHub, Inc., 88 Colusa Street, San Francisco, CA 94110, EE. UU.). GitHub procesa datos técnicamente necesarios (en particular direcciones IP) en el marco de la entrega del sitio web. La base jurídica es el art. 6 apdo. 1 lit. f RGPD. Para las transferencias de datos a los EE. UU., GitHub se basa en el Marco de Privacidad de Datos UE-EE. UU.

5. Tipos de letra

Este sitio web utiliza exclusivamente tipos de letra alojados localmente. No se cargan tipos de letra de proveedores externos (p. ej. Google Fonts). Por lo tanto, no se transmite ninguna dirección IP a terceros.

6. Contacto por correo electrónico

Si nos contacta por correo electrónico, sus datos se almacenarán con el fin de procesar la solicitud y para posibles preguntas de seguimiento. La base jurídica es el art. 6 apdo. 1 lit. b o f RGPD. No se transmiten datos a terceros.

7. Sus derechos

Usted tiene los siguientes derechos con respecto a sus datos personales:

Derecho de acceso (art. 15 RGPD), derecho de rectificación (art. 16 RGPD), derecho de supresión (art. 17 RGPD), derecho a la limitación del tratamiento (art. 18 RGPD), derecho a la portabilidad de los datos (art. 20 RGPD) y derecho de oposición al tratamiento (art. 21 RGPD).

Para ejercer estos derechos, póngase en contacto con la dirección de correo electrónico indicada anteriormente.

8. Derecho a presentar una reclamación ante la autoridad de control

Tiene derecho a presentar una reclamación ante una autoridad de control de protección de datos sobre el tratamiento de sus datos personales. La autoridad competente es el Comisionado Estatal de Baviera para la Protección de Datos, Wagmüllerstrasse 18, 80538 Múnich, Alemania.

9. Vigencia de esta política de privacidad

Estado: abril de 2026. Nos reservamos el derecho de modificar esta política de privacidad para que cumpla siempre con los requisitos legales vigentes.
